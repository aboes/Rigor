HINWEIS

    Lieg diese Datei - also ein zweites README - dem Programm bei, so handelt es sich um die
    Standalone-Version von "Alex' Rigor".
    Das bedeutet, dass alle zur Programmausführung notwendige Daten schon mitgeliefert werden.
    Insbesondere muss die Sprache Python 3 nicht installiert sein (sie wird hier mitgeliefert)
    um "Alex' Rigor" starten zu können.
    
    Informationen aus dem eigentlichen README sind deshalb obsolet. Namentlich der Abschnitt
    zwei "Was benötigt "Alex' Rigor" um zu starten?" ist ersatzlos zu streichen und im dritten
    Abschnit "Übersicht: Ordner- und Dateistruktur" kommen zusätzliche Dateien und Ordner
    hinzu.

WICHTIG

    Im Ordner "data_generation" findet sich nach wie vor eine ANLEITUNG wie man eigene
    Vokabelsätze für "Alex' Rigor" erzeugen und nutzen kann.
